## Shop Data Fields

Each shop in the registry contains:

| Field | Description |
|-------|-------------|
| Shop ID | Unique identifier |
| Location | City and country |
| Operator Address | Ethereum address of operator |
| Launch Date | When the shop opened |
| Status | Pilot, Active, Paused, or Closed |
| Cumulative Burned | Total SHAMA burned from this shop |
| Total Revenue | Historical revenue reported |

## Shop Status Types

<Info>
**Status Meanings:**
- **Pilot:** One of the first 4 team-operated shops
- **Active:** Operating shop in good standing
- **Paused:** Temporarily closed (maintenance, relocation)
- **Closed:** Permanently closed
</Info>
